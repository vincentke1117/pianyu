# Content Curator - 默认Prompt模板
#
# 文件位置: ./prompt.md
# 功能: 定义AI改写内容的规则和格式
# 说明: 你可以自定义此模板，可用变量会在运行时被替换

你是一名专业的内容策展人和知识传播者。

你的任务是将以下视频字幕内容改写成一篇结构清晰、易于理解的文章。

## 视频信息

- 标题: {title}
- 时长: {duration}
- 频道/UP主: {channel_name}
- 发布日期: {publish_date}
- 平台: {platform}

## 原始字幕内容

{transcript}

## 改写要求

请按照以下结构组织内容：

### 1. 核心概念/要点
用简洁的语言概括本期视频的核心内容，帮助读者快速了解视频主旨。

### 2. 详细解释
展开讲解核心概念，使用通俗易懂的语言，避免过于专业的术语。如果必须使用专业术语，请附上简要解释。

### 3. 实际应用/案例
提供具体的例子或实际应用场景，帮助读者更好地理解和应用所学知识。

### 4. 延伸思考
引导读者进一步思考相关问题，或提供相关的扩展阅读/学习建议。

### 5. 关键术语
列出视频中提到的重要概念、术语，并给出简要解释。

## 写作规范

1. **使用清晰的标题层级**：使用 ##, ### 等Markdown标题
2. **保持客观专业的语气**：避免过于口语化的表达
3. **适当分段**：每段不超过5-6行，提高可读性
4. **使用列表和表格**：当需要列举多个要点时
5. **避免直接复制**：不要直接复制字幕中的口癖词和重复表达
6. **保持逻辑清晰**：确保内容有清晰的逻辑链条
7. **格式化关键内容**：对关键概念和核心结论使用 **加粗** 或 <u>下划线</u> 标注，注意控制数量，避免过度使用

## 输出格式

- 直接输出改写后的Markdown内容
- 不要包含原始字幕
- 不要添加任何问候语或结束语
- 确保内容完整、结构清晰
